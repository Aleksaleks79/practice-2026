## Задание

Настроить систему контроля версий Git (репозиторий на GitHub/GitVerse, освоение базовых команд, регулярные осмысленные коммиты). Подготовить все проектные материалы в формате Markdown. Разработать статический веб-сайт о проекте по дисциплине «Проектная деятельность», используя только HTML и CSS (или, при желании, генератор Hugo). Сайт должен содержать: главную страницу с аннотацией, разделы «О проекте», «Участники» с описанием личного вклада, «Журнал» (минимум три публикации о прогрессе) и «Ресурсы» с полезными ссылками. Обязательно использовать графические и мультимедийные материалы.

## Вариативная часть задания

Разработать мобильное приложение дополненной реальности с использованием связки Unity 3D и Vuforia Engine. Необходимо реализовать маркерную AR-систему, в которой в качестве отслеживаемого изображения-маркера выступает банкнота номиналом 100 рублей. При наведении камеры устройства на данную купюру в реальном времени должно происходить распознавание и последующее отображение на экране трёхмерного объекта «стаканчик» (3D-модель). Приложение обязано обеспечивать интерактивный просмотр появившегося объекта: пользователь должен иметь возможность рассматривать стаканчик с разных сторон (вращение, приближение/отдаление) средствами сенсорного управления. Объект должен быть корректно привязан к положению купюры в пространстве, устойчиво отслеживаться при перемещениях камеры и пропадать при выходе маркера из поля зрения.
